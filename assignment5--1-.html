Questions

  1. Use read_csv to import the data. Show the number of NAs for each columns.
library(tidyverse)
df <- read_csv("https://bryantstats.github.io/math421/data/adult_census_missing.csv")
  1. Using function aggr the VIM package to plot the number of NAs for each column.
library(VIM)
aggr(df)

  1. Find other forms of missing values. Hint: You can use the table function to check if there are suspicious categories (Unknown, for example) in a variable. What all other forms of missing values shown in the data.
table(df$workclass)

##               12            22664             1110             2537 
##        State-gov          Unknown      Without-pay

According to the df workclass table, there are other forms of missing values under the categories, Not Known, Unknown, and ?

  1. Replace all the forms of missing values found to NA
df <- df %>% na_if('Unknown') %>% na_if('Not Known') %>% na_if('?')
  1. Replot the number of NAs for each column.

  1. Approach 1 to handle NAs: remove all rows that have any NAs. Save the dataset after removing as a different data. The original data is unchanged (still have NAs). How many rows left after removing?
df2 = drop_na(df)

Rows left: 29,865

  1. Approach 2 to handle NAs: Fill all the NAs with the previous or next value. (Hint: Using fill function). Save the dataset after filling as a different data. The original data is unchanged (still have NAs).
df3 = df %>% fill(workclass, .direction = 'updown')
  1. Approach 3 to handle NAs: For numeric variable, replace the NAs by the median. For categorical variables, replace the NAs to the majority.
df$worklass <- replace_na(df$workclass, 'Public')
